# Como executar o projeto

## Front end

```bash
# clonar repositório
git clone https://github.com/Alexz96/move-it.git

# entrar na pasta do projeto
cd move-it/moveit-next

# instalar as dependências
npm install ou yarn install

# executar o projeto
yarn dev
```
